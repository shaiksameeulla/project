inner join : matching records with the oncondition
left join = inner join +  any additional  records left table

right join = inner join +  any additional  records  from the right table
full join = inner join +  any additional  records left table + any additional  records  from the right table

join == inner join
left join = left outer join

right join = right outer join

full join = full outer join
